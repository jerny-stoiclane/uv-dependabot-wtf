# `hcm` - app.armhr.com

## Getting started

```bash
yarn install
```

Then copy the `.env.template` to `.env` and fill in the values. These values can be found in the Armhr 1Password Team.

## Running the app

```bash
yarn start
```

will start the HCM application at http://localhost:3200

### Commands

- `start`: runs the application on `localhost:3200`
- `build`: creates the production build version
